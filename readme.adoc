== Installation
* VRChat Creator Companion: https://squirrelbite.github.io/vpm/
* Unity Package Manager: `Window` -> `Package Manager` -> `+` -> `Add package from git URL...`
Add the following URL: `https://github.com/emperorofmars/stf_unity.git#upm`
